
# Instal script
```
apt update && apt install curl wget shc bzip2 -y && wget -q -O setup.sh "https://raw.githubusercontent.com/rifstore/RIFQIVIP01/main/setup.sh" && chmod +x setup.sh && ./setup.sh
```
